# Smart-Property-Platform
这款智慧物业微信小程序，为业主与物业搭建便捷桥梁。业主能线上缴纳物业费、查询水电燃气用量及费用，还可一键报修家中设施问题，实时跟踪维修进度。对小区公共事务，能参与线上投票表达意见。物业人员可借此快速发布通知公告、管理业主信息、安排维修任务等，实现高效办公。智慧物业小程序，让小区管理更智能，生活更省心 。
